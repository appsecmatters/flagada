Validate

Validate a flag submission

Checks if the SHA256 hash of the submitted value matches a known flag. On first valid submission, triggers the application workflow (GitHub invite, etc.) and marks the flag as FOUND.

Request Body schema: application/json
required
value
required
string^[0-9a-fA-F]{64}$

64-character hex string (SHA256 of the flag)

owner
required
string

GitHub username of the researcher
